# 🌱 Crop Prediction System (ML Web App)

## 📌 Overview
The Crop Prediction System is a Machine Learning–based web application that predicts the most suitable crop based on soil nutrients and environmental conditions.  
The model analyzes inputs like Nitrogen, Phosphorus, Potassium, temperature, humidity, pH, and rainfall to generate accurate crop recommendations.

---

## 🚀 Live Demo
🔗 https://crop-prediction-ml-g0q0.onrender.com/

---

## ⚙️ Features
- 🌾 Predicts best crop using ML model  
- 📊 Takes real-time soil & climate input  
- 🌐 Flask-based interactive web interface  
- 📱 Responsive & clean UI design  
- 🔄 Reset option for quick testing  

---

## 🧠 Technologies Used
- Python  
- Scikit-learn  
- Flask  
- HTML, CSS  
- Render (Deployment)

---

## ⚙️ How It Works
1. User enters soil nutrients and climate values.  
2. Data is sent to the trained ML model.  
3. Model predicts the best crop.  
4. Result is displayed instantly on the web page.
